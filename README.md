# bitwise-or

Bitwise OR (`|`) as a function. Just in case you ever need to pass the operation
to something else as a function.

## Example

``` javascript
var bitwiseOR = require('bitwise-or');

bitwiseOR(14, 9);
// => 15
```

## Installation

``` bash
$ npm install bitwise-or
```

## API

``` javascript
var bitwiseOR = require('bitwise-or');
```

### `bitwiseOR(x, y)`

Performs the `OR` operation on bits `x` and `y`.
